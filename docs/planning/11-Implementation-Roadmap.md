# 11-Implementation-Roadmap

## 1. v1 总排期（建议）

起始周：2026-03-09  
目标上线：2026-07-12

## 2. 周粒度里程碑

1. W1-W2：需求冻结与字段基线  
交付：phase0 文档定稿，验收口径确认

2. W3-W4：架构与数据层骨架  
交付：本地数据库模型、规则包加载器、日志骨架

3. W5-W7：PDF/DWG 解析与焊缝抽取  
交付：焊缝提取服务、置信度输出

4. W8-W9：人工确认工作台 + 主数据模块  
交付：焊缝编辑、PQR CRUD、焊工 CRUD

5. W10-W12：匹配引擎与冲突解释  
交付：国标/ASME 规则包、解释输出

6. W13-W14：模板系统与导出  
交付：预设模板、自定义映射、预览、Word/PDF

7. W15：许可证模块  
交付：在线激活、离线激活流程

8. W16-W17：集成测试与性能优化  
交付：缺陷收敛、性能报告

9. W18：UAT 与发布  
交付：安装包、发布说明、验收签字

## 3. 角色分工建议

- 产品经理：需求与验收管理
- 焊接专家：规则条款和字段口径
- 技术负责人：架构与风险收敛
- 后端：规则引擎、数据、许可证
- 桌面端：工作台、模板、导出交互
- AI：推荐与解释增强、降级策略
- QA：功能、回归、性能、UAT

## 4. v1.1 预排期

2026-07-13 至 2026-08-09（4 周）  
重点：局域网共享库、并发控制、基础权限
