# 09-NFR

## 1. 性能指标（v1）

- 冷启动 <= 8s
- 项目打开 <= 3s
- PDF 解析（20 页内）<= 60s
- DWG 解析（中等复杂度）<= 90s
- 匹配（100 焊缝/1000 PQR/500 焊工）<= 5s
- 工艺卡预览 <= 2s
- Word/PDF 导出 <= 10s

## 2. 容量指标

- 单项目支持 >= 500 条焊缝
- 主数据支持 >= 50,000 条记录（PQR+焊工）
- 审计日志保留 >= 180 天（可配置）

## 3. 稳定性指标

- 核心流程成功率 >= 99%
- 异常退出后可恢复最近自动保存点
- 自动保存默认 60s

## 4. 离线可用性

- 离线模式主流程可用（导入、确认、匹配、导出）
- AI 不可用时降级但不断流
- 授权在线校验失败走宽限期

## 5. 安全与审计

- 本地数据与许可证缓存加密
- 敏感字段脱敏展示
- 关键操作全量日志
- 导出与匹配记录 trace_id

## 6. 兼容性

- Windows 10/11 64-bit
- 最低分辨率 1366x768
- 导出文档兼容常见 Office 版本

## 7. v1.1 附加（多人共享）

- 并发用户目标：20
- 常规查询响应 <= 2s
- 并发写入冲突控制
- 断线重连与可执行提示
