# 10-Acceptance-v1

## 1. 验收目标

确认 v1 在单机环境下可完成端到端业务闭环，并满足可审计、可导出、可授权要求。

## 2. 验收环境

- Windows 10/11 64-bit
- PDF + DWG 测试图纸
- PQR、焊工资格、焊材库存样本库
- 标准包：`CN_GB`、`ASME_IX`

## 3. 功能验收清单

### 3.1 导入与提取

- PDF 导入成功并生成焊缝信息表
- DWG（复杂符号/剖视）可提取关键字段
- 未确认焊缝时禁止正式匹配

### 3.2 焊缝确认工作台

- 支持增删改、批量编辑、不确定项标记
- 支持变更历史追踪（before/after/operator/time/reason）

### 3.3 主数据模块

- PQR 支持增删改查与字段校验
- 焊工资格支持增删改查与有效期校验
- 字段配置变更可生效

### 3.4 焊材库与库存

- 焊材主数据支持增删改查（牌号、规格、批次、到期、库存）
- 支持库存导入与手动调整
- 库存不足与安全库存预警可正确触发
- 临期焊材可正确标识

### 3.5 匹配与解释

- 支持手动选择国标/ASME
- 返回 `match/partial/fail`
- 冲突解释包含字段、实际值、期望值、条款引用
- 返回 `trace_id` 与规则包版本
- 输出库存可执行性结论（可执行/需补料/不可执行）

### 3.6 库存策略

- `warn` 模式：库存不足不阻断匹配，但必须给 warning
- `strict` 模式：库存不足阻断匹配并返回 fail
- 库存冲突需包含建议（补料、替代批次、调整计划）

### 3.7 AI 增强

- AI 不推荐不合规项
- AI 故障时主流程可继续
- 低置信度建议不进入默认推荐位

### 3.8 模板与导出

- 预设模板可用
- 企业模板映射可保存版本
- 预览与 Word/PDF 导出字段一致
- 空值策略 `error/warn/allow` 生效

### 3.9 许可证

- 在线激活成功
- 离线 `licreq -> lic` 激活成功
- 过期、签名无效、设备不匹配可正确阻断

### 3.10 审计日志

- 导入、修改、匹配、导出、激活均有日志
- 日志包含 `time/action/operator/trace_id/result`
- 库存检查事件可追溯至批次与检查时间

## 4. 非功能验收

- 关键性能指标达成率 >= 95%
- P0/P1 缺陷为 0
- 离线核心流程全通过

## 5. 发布门槛

- 功能验收通过率 100%
- 审计追溯链路抽检通过率 100%
- 在线/离线激活均通过端到端验证
